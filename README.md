# 🛒 Online Retail Store Management (SQL Server)

## 📌 Project Overview

This project demonstrates the design and implementation of an **Online Retail Store Management System** using **Microsoft SQL Server**. It follows a complete data engineering and analytics workflow—from importing raw transactional data to designing a normalized database, optimizing performance, and generating actionable business insights through SQL.

The project showcases practical SQL Server concepts such as **database normalization (3NF), indexing, views, stored procedures, triggers, and advanced analytical queries**.

---

## 🚀 Key Features

* Imported **100,000+ retail order records** using `BULK INSERT`
* Designed a **3rd Normal Form (3NF)** relational database
* Established relationships using **Primary Keys** and **Foreign Keys**
* Optimized query performance with **Indexes**
* Created analytical **Views** for business reporting
* Developed parameterized **Stored Procedures**
* Implemented **Triggers** for automated audit logging
* Performed business analysis using advanced SQL queries and Window Functions
